## Positive or Negative

````java[]


import java.util.*;
public class Main
{
	public static void main(String[] args) {
	    Scanner s=new Scanner(System.in);
	    int n=s.nextInt();
	    if(n>0){
		System.out.println("positive");
	    }
	    else{
	      System.out.println("negative");  
	    }
	}
}

````

## Odd or Even

````java[[]

import java.util.*;
public class Main
{
	public static void main(String[] args) {
	    Scanner s=new Scanner(System.in);
	    int n=s.nextInt();
	    if(n%2==0){
		System.out.println("even");
	    }
	    else{
	     System.out.println("odd");   
	    }
	}
}
 ````
